Rest-API Kyselyt

Kirjat:

(URL-osoite: localhost:8080/api/book/)

Hae kaikki kirjat:

(GET) localhost:8080/api/book/

Hae kirja ID:n perusteella:

(GET) localhost:8080/api/book/{{id}}

Lisää kirja:

(POST) localhost:8080/api/book/{{kirjaryhmän_id}} 

{
    "title": "{{ Kirjan nimi }}",
    "author": "{{ Kirjan kirjoittaja }}"
}

Päivitä kirja:

(PUT) localhost:8080/api/book/{{kirjan_id}} 

{
    "title": "{{ Kirjan nimi }}",
    "author": "{{ Kirjan kirjoittaja }}",
    "bookgroup": {
    	"id": {{ uuden_kirjaryhmän_id }}
    }
}

Poista kirja:

(DELETE) localhost:8080/api/book/{{kirjaryhmän_id}}

Kirjaryhmät:

Hae kaikki kirjaryhmät:

(GET) localhost:8080/api/bookgroup

Hae kirjaryhmä ID:n perusteella:

(GET) localhost:8080/api/bookgroup/{{id}}

Lisää kirjaryhmä:

(POST) localhost:8080/api/bookgroup 

{
    "name": "{{ kirjaryhmän_nimi }}",
}

Päivitä kirjaryhmä:

(PUT) localhost:8080/api/bookgroup/{{id}} 

{
    "name": "{{ kirjaryhmän_nimi }}",
}

Poista kirjaryhmä (poistaa myös ryhmään liittyvät kirjat):

(DELETE) localhost:8080/api/bookgroup/{{id}}
