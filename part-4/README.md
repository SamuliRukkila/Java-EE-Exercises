# Part 4 assignments - Spring & Liferay

### #1 Sprint console-application (DI & AOP)

In this assignment we'll create a simple application where we demonstrate depositing and withdrawing money
from the account. We'll using loose dependency injection for the relationships between beans and AOP-bean
to log different actions in an outside module.

### #2 Spring MVC (Spring Boot) -tutorial

In this assignment we'll create a simple Spring MVC -application from the guide. We'll create traditional layers
like view, service and controller (repository). The main task in this tutorial is to answer some of the asked 
questions (I've answered for them in the code via comment-blocks).

### #3 Spring Transaction -tutorial

In this assignment we'll create a simple Spring MVC -application where emphasis is to try Spring's ```@Transactional``` -annotation. We'll create a simple bank-account -interface where one can send money to another's account. The main task
in this tutorial is to answer some of the asked questions (I've answered for them in the code via comment-blocks).

### #4 Spring Rest-API

In this assignment we'll create a simple Rest-API using Spring MVC + JPA (Hibernate). In this program one can perform
CRUD-operations on books and bookgroups. Additionally all endpoints are secured expect GET-requests.
