# Java EE: Exercises

Repository which contains all the exercises from Java EE -course in JAMK.<br />
Official course description [HERE](https://asio.jamk.fi/pls/asio/asio_ectskuv1.kurssin_ks?ktun=HTS30110&knro=&noclose=&lan=e&ark=).

## Exercises

All the exercises are distruped to 4 different segments (part-1, part-2 etc.). All of these segments will contain a brief documentation on the exercises.

* Part-1 -> *Servlets, JSP, JSTL & Beans*

* Part-2 -> *JSF & PrimeFace*

* Part-3 -> *JPA, Web services & EJB*

* Part-4 -> *Spring Framework & Liferay*
