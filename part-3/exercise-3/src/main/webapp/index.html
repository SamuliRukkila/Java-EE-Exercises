JPA REST-API

Tällä ohjelmalla ei ole käyttöliittymää - ota yhteys esim. Postmanilla tehdäksesi HTTP-kutsuja

Esimerkkikutsuja:

Hae kaikki asiakkaat:
  • (GET) http://localhost:8080/Customer-REST-API/customersapi/customer
Hae asiakas ID:n avulla:
  • (GET) http://localhost:8080/Customer-REST-API/customersapi/customer/{{:id}}
Poista asiakas ID:n avulla:
  • (DELETE) http://localhost:8080/Customer-REST-API/customersapi/customer/{{:id}}
Lisää asiakas:
  • (POST) http://localhost:8080/Customer-REST-API/customersapi/customer (application/JSON)
  • Esimerkki asiakkaan tiedoista:
{ "email": "esimerkki.esim@hotmail.com", "nimi": "Aapeli Aatukka", "osoite": "Mutatie 13B 93", "puhelin": "040 1212123", "salasana": "salainensalasana" }
